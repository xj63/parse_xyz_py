Parses an XYZ coordinate file into a vector of [f64; 3] arrays.

This function reads the contents of the specified file and parses it using the parse_xyz function.

ยงArguments

  • file_path - A value that can be borrowed as a Path, representing the path to the XYZ coordinate file.

ยงReturns

A Vec of [f64; 3] arrays, where each array represents a point in 3D space. Returns an empty Vec if the file is empty or contains no valid lines.

Panics if the file cannot be read or if the file content is not a valid XYZ coordinate format, according to the xyz.pest grammar.
